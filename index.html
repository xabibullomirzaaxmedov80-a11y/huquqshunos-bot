Mustaqil Telegram Botni ishga tushirish

Agar siz botni o'zingizning shaxsiy serveringizda doimiy ishga tushirmoqchi bo'lsangiz, quyidagi ko'rsatmalardan foydalaning:

  1. Node.js ni o'rnating:

    Tizimingizda Node.js o'rnatilgan bo'lishi kerak. Rasmiy saytdan yuklab oling.

  2. Bot loyihasini yuklang:

    Loyihamizda tayyorlangan `bot_script.js` faylini loyiha papkasidan oling.

  3. Kutubxonalarni yuklang: 
    npm install telegraf dotenv
  4. Botni ishga tushiring: 
    node bot_script.js
